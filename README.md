# Source and Version control
A  client has requested the development of a website that showcases their artwork for an art show that's coming to Auckland. The website should include the following features:
•	A title at the top of the page that says "The Canvas Collective"
•	Display of 3 pieces of artwork, showcasing the client's current collection
•	3  descriptions, one for each piece of artwork

•	A section dedicated to information about the artist, including a description and background information
•	A section dedicated to the location of the gallery, providing the address.
•	Utilization of Api.net for optimal performance and functionality.

Tech Stack:
Visual Studio Code
ASP.Net
Razor views
HTML
CSS
